# 世界漫游指南 - 架构文档

## 项目概述

**项目名称**: 世界漫游指南 (World Travel Guide) v1.3.0

**项目定位**: AI 驱动的互动小说引擎 (AI-Powered Interactive Fiction Engine)

**技术栈**:
- 前端框架: React 19 + TypeScript
- 构建工具: Bun (原生打包，Bun.build())
- 状态管理: Zustand + React Context (useReducer)
- 持久化: IndexedDB (存档) + localStorage (配置)
- AI API: OpenAI-compatible SSE 流式接口
- 样式: 纯 CSS + CSS 变量设计系统 + CSS Layers (base/layout/state/theme/print/a11y)

### 管线执行流程

![管线架构](diagrams/pipeline.svg)

1. **PromptAssemble** — 从 PresetPack 构建系统提示
2. **ContextPrepare** — 清理消息上下文，应用正则脚本
3. **VariableInject** — 注入当前变量状态到提示词
4. **WorldBookScan** — 扫描世界书条目并注入
5. **ModuleApply** — 应用世界模块（生存、经营、骰子等）
6. **APIRequest** — 发送 SSE 流式请求
7. **ResponseExtract** — 从响应中提取结构化数据
8. **VariableUpdate** — 更新变量状态

### 扫描引擎特性

- **正则关键词**：支持正则表达式匹配
- **选择逻辑**：selective 模式，主关键词 + 次关键词
- **排除关键词**：excludeRecursion 防止递归
- **递归扫描**：支持多层级递归扫描
- **分组互斥**：同组条目互斥激活
- **概率触发**：triggerChance 概率控制

### NPC 系统

- **创建流程**：useNpcCreate Hook 管理 NPC 创建
- **数据填充**：useNpcFill Hook 自动填充 NPC 数据
- **结构验证**：ensureNpcStructureDefaults 确保数据完整性
- **快照格式化**：formatSnapshotForMainAI 用于 AI 提示词

## 叙事记忆系统

### 管线架构

```
┌─────────────────────────────────────────────────────────────┐
│                   记忆管线 (memory/*)                        │
│  ┌───────────────┐  ┌───────────────┐  ┌───────────────┐   │
│  │ memoryPipeline│  │ useMemorySys  │  │ memoryStore   │   │
│  │ 六阶段管线     │  │ React Hook    │  │ Zustand Store │   │
│  └───────┬───────┘  └───────────────┘  └───────────────┘   │
│          │                                                  │
│  ┌───────▼───────┐  ┌───────────────┐  ┌───────────────┐   │
│  │ narrativeGraph│  │ narrativePars │  │ vectorUtils   │   │
│  │ 叙事图谱       │  │ 叙事解析器     │  │ 向量工具       │   │
│  └───────────────┘  └───────────────┘  └───────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

### 六阶段管线

1. **Write（写入）** — 从对话中提取叙事片段写入记忆库
2. **Summary（摘要）** — 对记忆片段进行摘要压缩
3. **Vector（向量化）** — 将记忆片段向量化用于检索
4. **Retrieve（检索）** — 根据当前上下文检索相关记忆
5. **Rerank（重排）** — 对检索结果进行重排序
6. **Compile（编译）** — 将记忆编译为可注入提示词

### 数据流

```
对话消息 → Write → 记忆片段 → Summary → 摘要
                                    ↓
                              Vector → 向量索引
                                    ↓
当前上下文 → Retrieve → 相关记忆 → Rerank → 编译 → 注入提示词
```

## 路由与导航

本项目**不使用 React Router**，导航完全由状态驱动。

### 顶层路由

`GameContext` 内部通过 `useReducer` 管理一个三屏状态机：

```
Screen: 'start' | 'settings' | 'game'
```

### Provider 嵌套层级

```
ErrorBoundary
  └── UISettingsProvider (主题/字体)
        └── GameProvider (导航状态)
              └── AppContent (switch 分发三个 Screen)
```

## 依赖规则

- 上层可以依赖下层，下层不能依赖上层
- 模块之间不能相互引用
- 上层通过接口调用下层
